Sobre este proyecto


Este sitio web fue desarrollado como parte de la Práctica 7: Introducción a CSS del curso de Programación de Aplicaciones Web. El objetivo principal es conocer e implementar CSS3 en sitios HTML, aplicando estilos y diseño visual para mejorar la experiencia del usuario.


Durante el desarrollo de esta práctica se implementaron técnicas de estilizado CSS como: configuración de fuentes, colores, márgenes, efectos hover, y diseño de menús de navegación interactivos.
